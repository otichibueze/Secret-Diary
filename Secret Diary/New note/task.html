Description

In the first stage, you should implement the basic functionality of the diary. Users should be able to enter some text they want and save it with a button. After this operation, the saved text should appear in the diary.

Objectives
  1. Create the following components of the user interface with the appropriate features:
    • An Activity named MainActivity
    • An EditText field with id attribute set to etNewWriting and hint attribute set to "Dear Diary...";
    • A Button with id btnSave and text "Save";
    • A TextView with id tvDiary that shows the saved text after click on save button;
  2. Add the following functionalities to the application:
    • When Save button is clicked, the diary should display exactly the same text as users had entered;
    • If the diary already contains some text, overwrite it again and again;
    • After every saving operation, the input field should be cleared;
  3. Handle exceptional usage of application:
    • When Save button is clicked, and input is either empty or contains only whitespace characters then input should not be saved, and a Toast message should warn users with the following text: "Empty or blank input cannot be saved";

Obs: Do not use mutation in global variables because tests do not support that.
