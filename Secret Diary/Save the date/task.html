Description

In second stage, add two more functionalities to your application. First, the diary should save all notes, not only the last one. Also, users may want to see when each note is created, so you need to attach the date and time when a message was added.

Objectives
  • Change the functionality of Save button. When users clicks on it, append a new note to the top of the diary with the timestamp, which is the date and time of creation;
  • The date and time come first with the following format: yyyy-MM-dd HH:mm:ss. The saved text should appear in the following line;
  • Timestamp should take user timezone into account;
  • Separate each note with an empty line;

Obs: The passage of time will be emulated on tests. There are several time related apis in java/kotlin/android environment, but tests time emulation won't guarantee to be compatible with all of them for this project.

You can use Clock.System.now() which will be guaranteed to be compatible with tests time emulation.
